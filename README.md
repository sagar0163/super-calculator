# Super Calculator

A powerful, feature-rich calculator that solves complex mathematical problems. From basic arithmetic to advanced calculus, statistics, and financial calculations.

## Features

- Basic & Scientific Operations
- Complex Number Support
- Matrix Operations
- Statistical Analysis
- Financial Calculations
- Unit Conversions
- Equation Solver
- Programmatic API & CLI

## Installation

```bash
npm install
npm link
```

## Usage

```bash
calc 2+2
calc "sin(45)"
calc --solve "2x + 5 = 15"
```
